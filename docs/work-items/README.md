# Work Items

This directory stores a chronological stream of non-trivial work and its artifacts.

Each work item lives in its own directory:

```txt
docs/work-items/NNN-<type>-<short-slug>/
```

Allowed task types:

- `feat`
- `change-request`
- `bug`
- `chore`
- `docs`

Recommended files inside a work item directory:

- `spec.md`
- `plan.md`
- `test-plan.md`
- `e2e-scenarios.md`
- `artifacts/`

Use `artifacts/` for screenshots, recordings, logs, or other evidence that helps verify the work.
